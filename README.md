# Concesionario Online

### Descripción
Este proyecto es un concesionario en línea especializado en la venta de motocicletas Honda. Ha sido desarrollado como proyecto de final de trimestre para la asignatura de Lenguaje de Marcas. El objetivo principal es ofrecer un catálogo visual y organizado de los modelos de motocicletas Honda, con secciones específicas para explorar las diferentes categorías de motos y opciones de contacto.

El proyecto incluye varias páginas funcionales accesibles desde el menú principal, así como enlaces decorativos que serán implementados en versiones futuras.

---

### Contenido del proyecto

#### Páginas accesibles:
1. **Inicio:** Página principal del concesionario.
2. **Modelos:**
   - **Adventure:**
     - Africa Twin
     - CB500X
     - NC750X
     - NX500
     - X-ADV
     - XL750 Transalp
   - **Custom:**
     - CMX500 Rebel
     - CMX1100 Rebel
     - CL500
   - **Scooter:**
     - PCX125
     - Forza 125
     - Forza 350
     - Forza 750
   - **Street:**
     - CB500F
     - CB650R
     - CB750 Hornet
     - CB1000R
   - **SuperSport:**
     - CBR500R
     - CBR600RR
     - CBR650R
     - CBR1000RR-R Fireblade
   - **Touring:**
     - NT1100
     - Gold Wing
3. **Contacto:** Información para contactar con el concesionario.
4. **Login:** Página para que los usuarios registrados accedan a su cuenta.
5. **Registro:** Formulario para que nuevos usuarios se registren.

#### Estructura del proyecto:
- **assets:** Contiene imágenes y videos organizados por modelo y categoría.
- **styles:** Hojas de estilos CSS personalizadas para las diferentes páginas.
- **scripts:** Archivo JavaScript para funcionalidades interactivas.

---

### Características adicionales
- **Responsividad:** Diseño adaptable para su visualización en dispositivos móviles y escritorio.
- **Multimedia:** Imágenes y videos de alta calidad para cada modelo de motocicleta.
- **Interactividad:** Navegación intuitiva y elementos visuales destacados.

---

### Notas importantes
Algunos enlaces en el menú son decorativos y no están implementados completamente en esta versión del proyecto. Esto incluye enlaces hacia secciones como configuradores o galerías avanzadas.

---

### Requisitos para ejecución local
1. Clonar el repositorio.
2. Abrir el archivo `hondamotorsport.html` en un navegador web.
